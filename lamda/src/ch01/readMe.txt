람다식 (Lamda expression) -> 람다 표현식

함수형 프로그래밍과 람다식

자바 : 객체 지향 프로그래밍
기능을 수행하기 위해서 객체를 만들고 멤버 변수, 메소드를 수행하게 코드를 설계한다.

함수의 구현과 호출만으로 프로그래밍 방식을 지원하는 (람다 표현식이라고 한다.)

함수형 프로그래밍(FP Functional Programing)

함수형 프로그래밍은 순수 함수(Pure Function)를 구현하고 호출함으로써
외부 자료에 부수적인 영향(side effect)을 주지 않도록 구현하는 방식입니다.

즉, 매개변수만을 사용하여 만드는 함수를 순수 함수라고 한다.

-> 실제 많이 사용하는 표현식
